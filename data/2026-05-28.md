### [1] [Non-invertible symmetry enriched string net topological orders](https://arxiv.org/abs/2605.28794)
*Luisa Eck,Peter Huston,Kyle Kawagoe,David Penneys*

Main category: cond-mat.str-el

TL;DR: 本文提出了非可逆对称富集拓扑序（NI-SETO）的定义，并通过弦网模型和任意子凝聚两种方式实现，利用酉融合范畴的相对中心描述所有NI-SETO，并举例分析了对称性作用。


<details>
  <summary>Details</summary>
Motivation: 已有研究多关注可逆对称性下的SETO，非可逆对称性下的拓扑序定义和实现尚不清晰，需要系统构建NI-SETO的理论框架。

Method: 采用全包含的酉融合范畴（UFC）和任意子凝聚两种方法，利用UFC的相对中心构建NI-SETO，并在弦网模型和3D Walker-Wang模型中实现，通过管代数技术分析对称性作用。

Result: 成功定义了NI-SETO，证明所有NI-SETO均可通过弦网模型或Walker-Wang模型实现，给出了多个实例并计算了任意子和对称缺陷上的定性对称性作用。

Conclusion: NI-SETO理论为理解非可逆对称性下的拓扑序提供了统一框架，UFC相对中心是其核心数学结构，管代数是分析对称性作用的有效工具。

Abstract: We propose a definition of a non-invertible symmetry enriched topological order (NI-SETO), and we implement our definition for string net models. We do so in two ways, using full inclusions of unitary fusion categories (UFCs), as well as anyon condensation. In both cases, the NI-SETO is a relative center of UFCs. All NI-SETOs can be realized in either model, where we can use enriched UFCs to get chiral examples on the boundary of a 3D Walker-Wang model representing the anomaly. We describe several examples of NI-SETOs and compute the qualitative symmetry action on anyons and symmetry defects using tube algebra techniques.
